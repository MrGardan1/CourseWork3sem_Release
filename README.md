
                                  Инструкция для пользователя

Для распаковки и использования данного приложения, требуется выполнить данные действия:

1) Открыть терминал и выполнить команду "unzip CtfApp_Release.zip" для распаковки папки.

2) Выполнить команду в терминале "cd CtfApp_Release", т.е. перейти в папку.

3) Выполнить команду в терминале "sudo ./run.sh" для того, чтобы скрипт мог выполнить 
   предварительные настройки.

4) Готово. Можно запускать исполняемый файл "./CtfApp".
