CTF Обучающее Приложение - Инструкция по запуску
Описание
Геймифицированное приложение для обучения основам CTF (Capture The Flag). Разработано на C# с использованием Avalonia UI и PostgreSQL.


# Инструкция по запуску
  
  # Шаг 1. Распакуйте архив
   unzip CtfApp_Release.zip
   cd CtfApp_Release

  # Шаг 2. Выдайте права на выполнение (если требуется)
   chmod +x run.sh

  # Шаг 3. Запустите приложение
   sudo ./run.sh


# При первом запуске:

   Автоматически установятся необходимые системные библиотеки
   Если Docker не установлен, скрипт установит его автоматически
   После установки Docker потребуется перезапустить терминал и запустить скрипт повторно

  # Шаг 4. Работа с приложением

   После успешного запуска:

1) Зарегистрируйте нового пользователя или войдите под существующим

2) Решайте задачи и зарабатывайте баллы

3) Отслеживайте свой прогресс в таблице лидеров

  # Шаг 5. Завершение работы

   Просто закройте окно приложения. База данных автоматически остановится.




# HELP: Решение проблем

Проблема: "Permission denied"
Решение: Запускайте скрипт с sudo:

sudo ./run.sh
Проблема: "Port 5432 already in use"
Решение: На компьютере уже запущен PostgreSQL. 

Остановите его:
sudo systemctl stop postgresql
sudo ./run.sh

Проблема: "Docker не запускается после установки"
Решение:

1) Закройте терминал (Ctrl+D)

2) Откройте новый терминал

3) Вернитесь в папку: cd ~/Путь/к/CtfApp_Release

4) Запустите снова: sudo ./run.sh

Проблема: Приложение не запускается (графические ошибки)
Решение: Установите графические библиотеки вручную:

sudo apt-get update
sudo apt-get install -y libice6 libsm6 libfontconfig1 libx11-6
sudo ./run.sh


Ручной запуск (для опытных пользователей)
Если автоматический скрипт не работает, можно запустить вручную:


# 1. Запустить базу данных
sudo docker compose up -d

# 2. Подождать 10 секунд

# 3. Запустить приложение
./CtfApp

# 4. После завершения остановить базу
sudo docker compose down
